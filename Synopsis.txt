Title Of The Project:
DELOITTE DESIGNATION

Introduction:
This project gives us Designation of freshers of Deloitte. This helps the company to make work easy of deciding Designation of fresher. With help of Eduction details, interview score and skills Designation of fresher will be decided.

System Requirement:
Windows 8,8.1,10
RAM 4GB and more
Chrome,FireFox and latest version of web browser
Internet Required

Software Requirement:
Java
Notepad++
Oracle
Spring Tool 4



Working:
Firstly user/admin need to login or signup.
If it is admin then redirected to admin menu page.
Admin shall select either Add new fresher or Change details of new fresher or Remove a fresher or Display all the fresher.
If Add new fresher then Personal information of fresher is added then Qualification information is added and then Designation is decided.
If change details of fresher then it will be directed to Personal information page then Qualification information page .
If Remove Fresher then directed toremove fresher page.
If Display all fresher then every freshers details and designation is displayed.
If it is fresher login then Userid and password is asked and the fresher will be directed to job description page.
